Задание было таковым: Написать программу которая выводит информацию на странице в таблице на по нажатию кнопки (Сделать запрос). Стиль страницы или использование фреймворка и инструментов по желанию. Единственное что сам запрос к API ресурса https://swapi.dev/, должен быть написан на python c указанием версии. 

1. Запросить данные персонажей  с https://swapi.dev/.
2. Вывести данные в таблице на странице в браузере. 
3. Проект выгрузить на GitHub и дать ссылку.

 В итоге, я создала простой проект на джанго на виртуальной среде для откладки. Нажимая на кнопку сделать запрос на главной странице выходить 10 людей из Вселенной Звездных войнов. Они получены из указанного выше ресурса.
